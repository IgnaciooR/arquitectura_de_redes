FTP¶

File Transfer Protocol.

Por hacer

Hacer

  • Va sobre TCP.
  • Permite la transferencia de archivos bidireccional entre un cliente y un servidor.
  • La sesión se establece de la misma forma que una sesión Telnet. Después se hace otra conexión para la transferencia de datos.
